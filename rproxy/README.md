rproxy configuration maker
================================

To make ***rproxy*** configuration easier and modulized.

NOTE:

> If this is firt time **make install**,
> please don't forget to backup the original ***rproxy*** copy.

For production:

    :::bash
    make clean install

For beagle site:

    :::bash
    make beagle install

For development site:

    :::bash
    make dev install
